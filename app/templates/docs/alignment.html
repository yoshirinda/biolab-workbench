Sequence Alignment Documentation

Overview

Multiple sequence alignment (MSA) is a fundamental step in comparative sequence analysis. It identifies conserved regions across homologous sequences and is essential for phylogenetics, protein structure prediction, and functional annotation.

Tool Comparison
Tool Speed Accuracy Best For
MAFFT Fast High General purpose, large datasets
ClustalW Moderate Good Classic tool, educational use
MUSCLE Fast High Medium-sized datasets
MAFFT Parameters
Parameter Default Description Recommendation
Algorithm Auto Alignment algorithm selection
  • Auto: Let MAFFT choose
  • L-INS-i: Most accurate, slow
  • G-INS-i: Global alignment
  • E-INS-i: For sequences with domains
  • FFT-NS-2: Fast, for large datasets
Max Iterations 1000 Number of refinement iterations
  • 0: No iteration (fastest)
  • 1000: High accuracy
Gap Open Penalty 1.53 Penalty for opening a gap Increase for fewer, longer gaps
Gap Extension Penalty 0.123 Penalty for extending a gap Increase for shorter gaps
--auto Off Automatic algorithm selection Enable for unknown data characteristics
ClustalW Parameters
Parameter Default Description
Scoring Matrix BLOSUM
  • BLOSUM: Best for proteins
  • PAM: For divergent sequences
  • GONNET: Alternative protein matrix
Gap Open 10 Penalty for opening a gap
Gap Extension 0.1 Penalty for extending a gap
MUSCLE Parameters
Parameter Default Description
Max Iterations 16 Maximum number of refinement iterations
Diagonal Optimization Off Use diagonal optimization for speed
Example Commands
MAFFT with high accuracy:
mafft --maxiterate 1000 --localpair sequences.fasta > alignment.fasta
MAFFT auto mode:
mafft --auto sequences.fasta > alignment.fasta
ClustalW:
clustalw -INFILE=sequences.fasta -OUTFILE=alignment.fasta -OUTPUT=FASTA
MUSCLE:
muscle -in sequences.fasta -out alignment.fasta
Quick Reference
Choosing the Right Tool:
  • Large dataset (>500 seq): MAFFT FFT-NS-2
  • High accuracy needed: MAFFT L-INS-i
  • Medium dataset: MUSCLE or MAFFT auto
  • Learning/teaching: ClustalW
Conservation Scores:
  •   90-100%: Highly conserved
  •   70-90%: Conserved
  •   50-70%: Moderately conserved
  •   <50%: Variable
Tips:
  • Remove redundant sequences before alignment
  • Check alignment manually for obvious errors
  • Consider trimming gappy regions
