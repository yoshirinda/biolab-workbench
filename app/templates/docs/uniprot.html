UniProt Search Documentation

Overview

UniProt (Universal Protein Resource) is a comprehensive database of protein sequences and functional information. This module allows you to search UniProt, filter by taxonomy and review status, and download sequences in FASTA format.

Search Parameters
Parameter Options Description
Search Query Text Keywords to search (e.g., protein name, gene name, function)
Taxonomy Filter Dropdown Filter by organism (Viridiplantae, Arabidopsis, etc.)
Database All / Reviewed / Unreviewed
  • Reviewed (Swiss-Prot): Manually curated, high quality
  • Unreviewed (TrEMBL): Automatically annotated
Max Results 1-5000 Maximum number of entries to retrieve
Header Format Gene_Species_ID / Standard Format for FASTA sequence headers

Workflow
  1. Enter search keywords (e.g., "ACO oxidase", "kinase")
  2. Select taxonomy filter to narrow results
  3. Choose database type (Reviewed recommended for quality)
  4. Click Search to retrieve entries
  5. Review results in the table, select entries of interest
  6. Click Download Sequences to get FASTA file
Tips
  • Use Reviewed database for curated, high-quality sequences
  • Combine keywords with taxonomy for specific results
  • Use Gene_Species_ID header format for phylogenetic analysis
  • Select multiple entries using checkboxes before downloading
