Full Pipeline: HMM Search → BLAST Alignment → Length Filtering → Multiple Sequence Alignment → ClipKIT Trimming → IQ-TREE Tree Building → Tree Visualization
Usage Instructions:
  • Each step can run independently: upload files or provide existing paths, no need to start from scratch.
  • 点击“清理序列头”预处理 FASTA, 避免特殊字符导致下游工具失败。
  • 参数建议: HMM e-value 1e-5; BLAST e-value 1e-3, Max target 100; ClipKIT kpic-gappy; IQ-TREE 模型 MFP + UFBoot 1000。
  • 所有结果都会给出独立下载链接; 如浏览器阻止, 复制链接在新标签打开。
